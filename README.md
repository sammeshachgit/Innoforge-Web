# 🎓 SECE Web Portal


A modern, responsive, and visually appealing college website . This project showcases academic information, courses, faculty, events, and more with a clean and user-friendly interface.

---

## 📌 Features

✨ Fully Responsive Design
✨ Modern UI with smooth animations (AOS, Animate.css)
✨ Structured Navigation Bar
✨ Dedicated Pages:

* Home
* About
* Courses
* Faculty
* Events
* Fees Structure

✨ Dropdown Academic Section
✨ Clean and maintainable code structure
✨ Cross-browser compatibility

---

## 🛠️ Tech Stack

| Technology                 | Purpose           |
| -------------------------- | ----------------- |
| HTML5                      | Structure         |
| CSS3                       | Styling           |
| Bootstrap                  | Responsive layout |
| JavaScript                 | Interactivity     |
| AOS (Animate on Scroll)    | Animations        |
| Animate.css                | UI effects        |
| Boxicons / Bootstrap Icons | Icons             |

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/sece-web.git
```

2. Navigate into the project:

```bash
cd sece-web
```

3. Open `index.html` in your browser:

```bash
start index.html
```
